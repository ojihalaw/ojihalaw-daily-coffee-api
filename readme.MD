## ðŸ”‘ Authentication

- Login with **email/password**
- Returns an `AuthResponse` containing:
  - **User info**
  - **Access token** (short-lived)
  - **Refresh token** (long-lived)
- **Access token** is used for API requests
- **Refresh token** is used to generate a new access token
